## Come pubblicare
1. Crea un repo su GitHub (es. `prototipi-montagna-servizi`).
2. Carica **tutto il contenuto di questa cartella** nella root del repo (drag & drop su github.com oppure `git push`).
3. Settings → Pages → Source: `main` / root → Save.
4. Dopo ~1 minuto le pagine sono su `https://<utente>.github.io/prototipi-montagna-servizi/piattaforma.html` e `.../presentazione.html`.

## PDF con link funzionanti
Esporta il PDF della presentazione **dalla pagina pubblicata** (aprila su GitHub Pages e stampa in PDF): i link diventano URL assoluti pubblici e funzionano ovunque.
